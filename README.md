# 🇵🇸 Smart Humanitarian Aid Tracker

A comprehensive, automated Java desktop application designed to manage, streamline, and optimize the distribution of humanitarian aid with fairness, speed, and strict transparency.

## 📌 The Core Inspiration
This project is not just an academic requirement; it is a solution born directly from the harsh daily realities of the Gaza Strip. Living through severe crises, I experienced firsthand the profound failures of manual aid distribution—long lines, chaotic paperwork, lost beneficiary data, and truly vulnerable families unable to access critical support. Driven by these challenges, I decided to apply my software engineering skills to build a functional tool designed for real-world relief operations, proving that technology can be a powerful instrument for social justice and human dignity (**Tech4Good**).

## 🚀 Key Features

*   **Intelligent Priority Assessment:** Features an automated algorithm that evaluates and ranks family needs based on critical metrics: family size, presence of individuals with special needs, and chronic medical conditions.
*   **Emergency Dispatch Module:** A separate, high-priority workflow to process urgent, life-threatening cases swiftly without queuing delays.
*   **Comprehensive Beneficiary Management:** Scalable registration forms to securely capture, update, and manage detailed household profiles.
*   **Robust Data Persistence:** Utilizes custom File I/O mechanisms to securely save, update, and retrieve records locally, preventing data loss during power outages or network disconnections.
*   **Intuitive Java Swing GUI:** A clean, accessible user interface optimized for fieldworkers and volunteers operating under high-stress environments.

## 💻 Technical Architecture & Concepts

This system is built entirely from scratch using **Java**, adhering strictly to production-level design principles:
*   **Object-Oriented Programming (OOP):** Implementation of **Inheritance** for different aid categories, **Polymorphism** for dynamic distribution logic, **Encapsulation** to secure family privacy, and **Abstraction** for core system components.
*   **Data Structures:** Uses **ArrayLists** and native **Arrays** for efficient runtime memory management and data manipulation.
*   **Defensive Programming:** Implements rigorous **Exception Handling** to prevent system crashes during faulty data entry.
*   **Persistence Layer:** Built using Java standard File Input/Output streams for lightweight, database-free local storage.
